该问题主要是由于 bootstrap.yml 中使用了 @ @ 变量，而在 target 目录中生成的 bootstrap.yml中的 @ @ 变量没有被正确替换。

# 解决办法

# 办法一

使用 mvn install 或 mvn package 命令重新编译工程。

# 办法二

在IDEA中，打开Maven面板，点击刷新按钮。如下图所示：
